# Day08 - 할인 및 배송비 계산기

## 목표
가격과 수량을 입력 받아 총액을 계산하고,
가격에 따른 할인 및 배송비를 동적으로 계산하여 출력하는 기능을 구현한다.

## 핵심 개념
- 입력 검증: 값의 유효성 체크 (빈값, 0 이하 값, 숫자 여부 등)
- 상태 관리: state 객체를 사용해 가격, 수량, 상태, 메시지, 할인, 배송비 등을 관리
- 상태 기반 렌더링: 상태에 따라 동적으로 화면에 반영
- 이벤트 처리: input과 click 이벤트를 통해 입력값을 처리하고, 결과를 렌더링

## 구조
- state: 현재 앱 상태 저장
- validateInput(): 입력값 검증
- applyDiscount(): 할인 적용
-	calculateShipping(): 배송비 계산
-	getMessageByStatus(): 상태에 따른 메시지 생성
-	render(): 화면 반영 전담

## 흐름
1.	가격과 수량을 입력받음
2.	입력 검증 후, 유효한 값일 경우 총액 계산
3.	총액이 50,000원 이상일 경우 할인 적용
4.	총액에 따른 배송비 계산 (50,000원 이상이면 무료배송)
5.	계산된 결과를 상태에 맞게 화면에 출력

## 배운 점
- 상태 기반 렌더링을 통해 화면 반영
- 핸들러 분리로 이벤트 관리
- 입력값 검증과 조건부 로직 처리
