# JS UI Lab + Mini Project

퍼블리셔에서 프론트로 확장하기 위해 만든 JavaScript 학습/실습 저장소입니다.  
**목표:** DOM/이벤트/비동기/상태관리 기본기 + UI 컴포넌트 구현 + 미니 프로젝트 완성

## 폴더 안내
- `js-basics/` : 문법/DOM/비동기 기초 실습
- `ui-lab/` : 모달/탭/드롭다운 등 UI 컴포넌트 구현
- `mini-project/mini-shop/` : 목록/상세/장바구니 미니 프로젝트(완성본)
- `archive/` : 과거 연습 코드 보관

## 실행 방법
대부분의 예제는 `index.html`을 브라우저로 열면 동작합니다.  
VS Code를 쓰면 Live Server 확장으로 실행하는 것을 권장합니다.

## 진행 로그(선택)
- YYYY-MM-DD: modal 구현(ESC 닫기/바깥 클릭 닫기)
- YYYY-MM-DD: tab 구현(키보드 포커스 개선)
