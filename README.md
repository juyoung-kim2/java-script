# JS UI Lab + Mini Project

실무 UI 구현 능력 향상을 위해 진행한 JavaScript 학습·연습 저장소입니다.
DOM 조작, 이벤트 처리, 비동기 흐름, 상태 관리 등 기본기를 익히고
이를 기반으로 다양한 UI 컴포넌트 및 미니 프로젝트를 구현했습니다.

## 폴더 안내
- `js-basics/` : 문법/DOM/비동기 기초 실습
- `ui-lab/` : 모달/탭/드롭다운 등 UI 컴포넌트 구현
- `mini-project/mini-shop/` : 목록/상세/장바구니 미니 프로젝트(완성본)
- `archive/` : 과거 연습 코드 보관

## 실행 방법
대부분의 예제는 `index.html`을 브라우저로 열면 동작합니다.  
VS Code 사용 시 Live Server 확장을 활용하면 더 편리합니다.
